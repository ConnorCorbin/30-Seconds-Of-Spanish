# Challenge Header component

Challenge Header features:

* Set question title
* Set question text

Consumed components:

* None

## Data Model

| Configurable Fields | Type | Component | Default | Description |
| ------------------- | ---- | --------- | ------- | ----------- |
| questionTitle | `string` | ChallengeHeader | | Question title |
| questionText | `string` | ChallengeHeader | | Question text  |
