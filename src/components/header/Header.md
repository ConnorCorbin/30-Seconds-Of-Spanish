Header features:

* Renders Image and title

Consumed components:

* Image

## Data Model

| Configurable Fields | Type | Component | Default | Description |
| ------------------- | ---- | --------- | ------- | ----------- |
| title1 | `string` | Header | | Text for title 1 |
| title2 | `string` | Header | | Text for title 2 |
| textColor | `string` | Header | | Text color for both title 1 and title 2 |
| imageUrl | `string` | Image | | Image alt tag |
| imageAltTag | `string` | Image | | Image alt tag |
