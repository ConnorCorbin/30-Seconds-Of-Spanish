Header features:

* Renders Image and title

Consumed components:

* Image

## Data Model

| Configurable Fields | Type | Component | Default | Description |
| ------------------- | ---- | --------- | ------- | ----------- |
| title1 | `string` | Header | | Text for title1 |
| title2 | `string` | Header | | Text for title2 |
| textColor | `string` | Header | | Text color for both title1 and title2 |
| imageUrl | `string` | Image | | Image alt tag |
| imageAltTag | `string` | Image | | Image alt tag |
