# Image Banner component

Image Banner features:

* Renders banner with image in center

Consumed components:

* Image

## Data Model

| Configurable Fields | Type | Component | Default | Description |
| ------------------- | ---- | --------- | ------- | ----------- |
| imageUrl | `string` | Image | | Image alt tag |
| imageAltTag | `string` | Image | | Image alt tag |
